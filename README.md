# Real Time Public Transport Tracking System (Java)

This project is a simple console-based simulation of a public transport tracking system designed for small cities.

## Features
- Add bus details
- Update bus location
- View all buses
- Simple menu-based interface

## Technologies Used
- Java
- OOP concepts
- ArrayList

## How to Run

1. Compile the program
   javac TransportTrackingSystem.java

2. Run the program
   java TransportTrackingSystem
