# Real Time Public Transport Tracking System for Small Cities

This project is a Java-based simulation of a public transport tracking system.

## Features
- Track buses in real time (simulation)
- View all available buses
- Check bus status
- Estimated arrival time
- Location tracking using latitude and longitude

## Technologies
- Java
- Object Oriented Programming
- ArrayList

## Use Case
This system can help passengers track buses in small cities where advanced GPS systems may not be available.
